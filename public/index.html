WhatsApp AI Bot Setup

Follow the steps below to connect your WhatsApp

Step 1: Create Meta Developer Account

1. Go to Meta Developers

2. Create a new account or log in

3. Create a new app and select "Business" type

Step 2: Set Up WhatsApp

1. From your app dashboard, click "Add Product"

2. Select "WhatsApp" and follow the setup process

3. Get your WhatsApp Phone Number ID from the API Setup page

4. Generate a permanent access token

Step 3: Configure Webhook

1. In WhatsApp settings, find "Webhooks"

2. Set the Webhook URL to:

Loading...

3. Create a Verify Token and save it

4. Subscribe to "messages" webhook field

Step 4: Set Environment Variables

Add these variables in your Vercel project settings:

  • WHATSAPP_TOKEN (your access token)
  • WHATSAPP_PHONE_ID (your phone number ID)
  • VERIFY_TOKEN (your webhook verify token)

Step 5: Test Your Bot

1. Send a message to your WhatsApp business number

2. The bot should respond automatically

3. Use /image command for image generation
